# java-cli-gradle-spring-jacoco-junit-car-data

## Description
A POC for spring app. Testing done with jacoco
plugin using JUnit4.

## Tech stack
- java
- gradle
  - spring
  - jacoco
  - junit4

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
